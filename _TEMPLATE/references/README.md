# References

>

> - Твой шаблон стратегического документа (на который опирается `launch-strategist`)
> - Методички по прогревам
> - Примеры офферов, которые сработали раньше
> - Референсы по вебинарам, марафонам
>
> Если шаблон универсальный для всех проектов — храни в `.claude/lessons/` или `.claude/references/` в корне репо, а сюда клади только то, что специфично для этого блогера.
