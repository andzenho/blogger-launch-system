Вызови агента `reels-writer` через Task tool.

**Режимы:**
- `write` — сценарии под прогрев (привязка к волнам W1–W4 из warmup-plan). Precheck: `warmup-plan.md` готов.
- `leadmagnet-batch` — сценарии для холодного трафика на лид-магнит. Только для `launch_model` = cold / hybrid. Precheck: `data/leadmagnet-funnel.md` готов.
- `competitor-analyze` — разбор конкурентных рилсов, обновление `research/reels-patterns.md` (сценарии в этом режиме НЕ пишутся).
- `adhoc` — произвольная задача.

Уточни у продюсера режим и сколько сценариев нужно.
