Вызови агента `event-writer` через Task tool.

**Precheck:**
- `research/event-brief.md` готов (launch-strategist Проход 2 завершён)
- `data/audience-profile.md`, `data/product-matrix.md`, `data/launch-strategy.md`, `data/warmup-plan.md` на месте

**Режимы event-writer** (спроси продюсера или укажи при вызове):
- `structure` — скелет вебинара с таймингами (первый шаг, всегда)
- `script` — speaker notes для каждого блока
- `slides` — визуал + текст + speaker notes для слайдов
- `pitch-only` — только питч-секция (оффер + Stack + close + Q&A)
- `adhoc` — произвольная правка существующего вебинара

По умолчанию начинай с `structure` → после утверждения продюсером переходи к `script` → `slides` → `pitch-only`.

Артефакты в `content/webinar/<NN>-<slug>/`.
