---
description: Пометить артефакт как проверенный. Перевод статусов my-review → client-review или client-review → done.
---

Продюсер хочет перевести один из артефактов в следующий статус. Действуй:

1. Спроси, какой артефакт и какой переход:
   - `my-review → client-review` — продюсер проверил, отдаём эксперту
   - `client-review → done` — эксперт утвердил, можно идти дальше
   - `my-review → правки` — продюсер даёт список правок, артефакт перезапускает автор-агент

2. Обнови поле `status` внутри артефакта (если оно есть в шапке).

3. Обнови `state.md` через orchestrator — поставь новый статус фазы + добавь запись в журнал событий.

4. Если переход разблокировал следующий шаг (например, audience-profile done → можно звать product-architect) — скажи об этом продюсеру, но не запускай автоматически.
