Вызови агента `sales-page-writer` через Task tool.

**Режимы** (уточни у продюсера, какой тип лендинга нужен):
- `registration` — лендинг под регистрацию на вебинар/эфир/марафон. Precheck: `research/event-brief.md` готов.
- `offer` — продающий лендинг после события. Два подрежима:
  - `direct-sale` — прямая продажа по кнопке
  - `application` — через заявку + прозвон
- `leadmagnet` — холодный трафик на PDF/чек-лист. Только для cold/hybrid. Precheck: `data/leadmagnet-funnel.md` готов.
- `thank-you` — страница подтверждения после опта.
- `adhoc` — правка существующего лендинга.

**Precheck для любого режима:**
- `data/audience-profile.md`, `data/product-matrix.md`, `data/launch-strategy.md` на месте

Артефакты в `content/landings/`.

Offer-лендинг обязательно идёт на client-review перед запуском.
