# /ig — прогрев в Instagram

Запускает агента `instagram-warmer`. IG работает на ежедневный эмоциональный контакт и удержание внимания через сторис. Лента и карусели усиливают важные точки (анонс, кейсы, разборы), Reels тянут холодный охват.

## Когда звать

- `data/warmup-plan.md` готов
- `audience-profile.md`, `product-matrix.md`, `launch-strategy.md` на месте

## Методология

Из `.claude/skills/instagram-warming/`:

- **Сторис** — одна сюжетная линия на день, 6–14 фреймов. Строится по технике **ЗАМЕТКА** (Зацеп → Активация → Метафора → Есть проблема → Твоё решение → Кейсы → Анонс)
- **Контент-банк** — 11 категорий материала (личное, метод, кейсы, разборы, повседневка, FAQ, мнение, закулисье, анонсы, опросы, провокации)
- **Пропорции по волнам** — что в W1 больше личного, в W3 — метода, в W4 — анонсов и социального доказательства
- **Reels** — места размечаются в плане, но сценарии пишет отдельный агент `/reels`

## Режимы

- **`stories-day`** — сторис-цепочка на один день (одна сюжетная линия, 6–14 фреймов)
- **`feed-post`** — подпись к посту ленты (обычно 1–2 поста в неделю)
- **`carousel`** — текст карусели (7–10 слайдов)
- **`promo`** — промо-пост под эфир/событие (анонс или финальное напоминание)
- **`castdev`** — режим касдев-прогрева. Требует `research/castdev-brief.md` + `state.md → castdev_form_url`. Работает **параллельно с** `/tg` в том же режиме
- **`adhoc`** — разовая правка, отдельный сторис-слайд, пост под запрос

## Precheck

Агент не стартует, если:
- `warmup-plan.md` отсутствует
- `audience-profile.md` не в `done`
- Для режима `castdev`: нет `castdev-brief.md` или `castdev_form_url` пустой

## Вход

- `data/warmup-plan.md`
- `data/audience-profile.md` — язык, боли, дневные триггеры
- `data/product-matrix.md`
- `research/project-digest.md` — тон, стиль подачи, особенности эксперта

## Выход

- `content/ig/stories/_index.md` — индекс сторис-дней
- `content/ig/stories/<NN>-<slug>.md` — цепочка на день
- `content/ig/feed/_index.md` — индекс постов ленты
- `content/ig/feed/<NN>-<slug>.md` — подпись к посту / карусель

Каждый артефакт содержит: дату, тип, структуру фреймов/слайдов, тексты, заметки по визуалу (что снимать / что показать).

## После завершения

- Продюсер сам проверяет каждый день перед публикацией
- Эксперт снимает по готовым сценариям — не импровизирует

Client-review **не обязателен** на этом уровне — коррекции идут в рабочем порядке.

## Типичные ошибки

- Пытаться писать рилсы через `/ig` — это задача `/reels`, там другая методология (вирусные хуки, 5 законов, шаблоны)
- Собирать сторис из случайных кадров без сюжетной линии — охваты падают после 3-го фрейма
- Игнорировать технику ЗАМЕТКА — цепочка без структуры теряет удержание
- В W4 зашивать слишком много продажных слайдов подряд — отписки. Нужно чередование с личным / кейсами
