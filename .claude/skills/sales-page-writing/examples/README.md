# Примеры посадочных страниц

Эталонные лендинги под все архетипы × ниши. Делится на **шаблонные** (синтетические, для демонстрации формата) и **реальные** (с метриками, добавляются по ходу работы).

## Что здесь сейчас

Шесть шаблонных примеров, покрывающих ключевые сочетания:

| Файл | Архетип | Подтип | Ниша | Длина |
|---|---|---|---|---|
| `registration_webinar_hard.md` | registration | webinar (короткий) | hard (маркетинг/продюсинг) | 2 экрана |
| `offer_direct-sale_hard.md` | offer | direct-sale | hard (запуски курсов) | 8 экранов |
| `offer_application_hard.md` | offer | application | hard (high-ticket наставничество) | 5 экранов |
| `offer_direct-sale_soft.md` | offer | direct-sale | soft (психология) | 7 экранов |
| `leadmagnet_minimal_hard.md` | leadmagnet | minimal (Hormozi) | hard | 1 экран |
| `leadmagnet_extended_soft.md` | leadmagnet | extended | soft (психология) | 3 экрана |

Фактура (имена, цифры, скриншоты) — **синтетическая**. Подставляется под реального эксперта при адаптации.

## Как работать с примерами

1. Агент `sales-page-writer` читает релевантный пример **перед первой страницей в новом проекте** — для калибровки тона, длины, ритма блоков
2. Не копировать блок «кейсы» дословно — эти имена и цифры выдуманы
3. Блок «🏆 Результат» в шаблонных примерах = `—`

## Формат имени файла

```
<архетип>_<подтип>_<ниша или короткая-тема>.md
```

Примеры реальных (когда появятся):
- `registration_webinar_hard_psyhologiya-2026-05-12.md`
- `offer_direct-sale_hard_course-flagship-2026-06-15.md`

## Добавление реальных кейсов

В реальные примеры добавляется блок «🏆 Результат» с метриками:

```markdown
## 🏆 Результат

- Опубликована: 2026-05-02
- Трафик за период: 4 200 уников
- Конверсия в опт-ин: 52%
- Конверсия из опт-ина в оплату (для offer): 4,8%
- Revenue / заявки: [цифра]
- Что работало / что переделали: [заметки]
```
