# Примеры сценариев рилс

Эта папка — банк эталонных сценариев: по одному на каждый ключевой формат и цель. Делится на две группы: **шаблонные примеры** (синтетические, для демонстрации формата новому проекту) и **реальные кейсы** (с метриками, пополняется по мере работы).

Шесть шаблонных примеров, покрывающих основные сценарии работы агента `reels-writer`:

Шаблонные примеры — **синтетические**. Фактура (Яндекс, 1200 консультаций, 1,2 млн за курс, Аня из Тюмени) подставляется под реальную фактуру эксперта при адаптации. Блок «🏆 Результат» в них пустой (`—`), потому что метрик нет.

## Как работать с шаблонными примерами

1. **`reels-writer` читает их**, когда сталкивается с новым проектом и нужен образец тона, длины, структуры для конкретного сочетания «волна × цель × ниша».
2. **Не копировать дословно** — фактура поставлена для иллюстрации, не для использования. Всегда адаптируй под реального эксперта.
3. **Soft-niche заметка в конце файлов** soft-примеров описывает специфику мягкой ниши (антикритик вместо дожима, отсутствие «быстрых результатов», реалистичные цифры). Её можно использовать как чек-лист при проверке сценария.

## Как добавлять реальные кейсы

Когда сценарий был снят и дал результат — кладём сюда с метриками. Формат имени файла:

```
<launch|leadmagnet>_<шаблон>_<короткая-тема>.md
```

Примеры:
- `launch_shablon1_perehod-v-agenstvo-2026-05-15.md`
- `leadmagnet_shablon8_nejro-tools-2026-06-03.md`

В шапке YAML — все поля как в шаблонных примерах (`status: example` заменяем на `status: published`). В конце файла блок «🏆 Результат» заполняем реальными цифрами:

```markdown
## 🏆 Результат

- Дата публикации: 2026-05-15
- Охват: 420к
- Просмотры: 580к (досмотр 38%)
- Сохранения: 12к
- Комментарии с кодовым словом: 2,4к
- Конверсия из комментария в подписчика бота: 67%
- Конверсия из подписчика в лид-магнит (скачал PDF): 82%
- Трипваер / флагман: (если считали)
- Заметки: [что сработало / что переделать в следующем]
```

Реальные кейсы с метриками весомее шаблонных примеров. Когда в нише накопится 3+ реальных кейсов по одному шаблону — шаблонные примеры для этой ниши можно удалить, оставив только проверенные.
